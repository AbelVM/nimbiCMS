# Function: isExternalLinkWithBase()

> **isExternalLinkWithBase**(`href`, `contentBase?`): `boolean`

Determine whether an href points outside of the provided contentBase.

## Parameters

### href

`string`

The href to test for being external.

### contentBase?

`string`

Optional base URL to use when resolving absolute paths.

## Returns

`boolean`

- True when the href points outside the provided content base.
