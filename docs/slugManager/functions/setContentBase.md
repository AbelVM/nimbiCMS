# Function: setContentBase()

> **setContentBase**(`contentBase?`): `void`

Set the content base URL (the runtime `contentPath`) and rebuild slug
maps and `allMarkdownPaths` relative to that base.

## Parameters

### contentBase?

`string`

Optional base URL where markdown content is hosted

## Returns

`void`

- No return value.
