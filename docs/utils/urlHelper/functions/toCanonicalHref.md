# Function: toCanonicalHref()

> **toCanonicalHref**(`href`): `string`

Convert a cosmetic or canonical href into the canonical `?page=` form.
Returns the canonical href string (e.g. `?page=slug#anchor`).

### href

`string`

href to convert

## Returns
