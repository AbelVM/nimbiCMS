# Function: renderNotFound()

> **renderNotFound**(`contentWrap`, `t`, `e`): `void`

Render a simple "not found" message into the provided container.
Placed immediately above export for TypeDoc.

## Parameters

### contentWrap

Container where the message will be rendered; may be null.

`HTMLElement` | `null`

### t

Translation function that accepts a key and returns a localized string.

`Function` | `null`

### e

Optional error whose message may be displayed to the user.

`Error` | `null`

## Returns

`void`
