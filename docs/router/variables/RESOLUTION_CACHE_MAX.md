# Variable: RESOLUTION\_CACHE\_MAX

> **RESOLUTION\_CACHE\_MAX**: `number` = `100`

Maximum number of entries stored in the resolution cache before oldest
entries are evicted. Mutable at runtime via `setResolutionCacheMax`.
