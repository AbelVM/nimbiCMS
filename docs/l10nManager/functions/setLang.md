# Function: setLang()

> **setLang**(`lang`): `void`

Switch the current UI language. Falls back to English if the requested
language is not available.

## Parameters

### lang

`string`

Language code to switch to (e.g. 'en', 'es').

## Returns

`void`

- No return value.
